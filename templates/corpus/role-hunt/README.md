# role-hunt/

Your career corpus. Maintain `watchlist.yml` (companies + filters) + `applications.yaml` (your pipeline)
by hand; the rest is generated:

- `watchlist.yml` — companies to scan + role filters.
- `applications.yaml` — the pipeline you're pursuing (you advance stages; the tool never applies).
- `discoveries/` — dated scan reports (`hn career openings --write`).
- `companies/` — per-company profiles (`hn career company-profiles --write`); the Hiring-Map deep-links.
