# travel/

Your travel corpus. Maintain `config/weights.yaml` (home base + preferences); the rest accretes:

- `config/weights.yaml` — home base, flight/lodging/event preferences.
- `trips/` — outbound trip folder-notes (the pipeline reads `trip:` frontmatter).
- `visits/` — inbound hosting visits.
- `reference/` — the key-dates / sports almanac the dashboard surfaces.
